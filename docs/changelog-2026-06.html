Changelog

June 2026

🔎

Search results as clean JSON

A single native binary that turns raw search-engine pages into structured JSON your agents can use — Google organic results, Google Maps places, DuckDuckGo and Bing. Pipe in raw HTML, get back titles, links, snippets and display links.

🧩

Bring your own scraper

Fetching is a plugin, not a lock-in. Any program that prints a page to stdout is a fetcher — curl, a headless browser, a paid API, or a cached file. open-serpapi never touches the network, so it has no secrets to leak and runs the same offline.

🛡️

Past the CAPTCHA wall

Included anti-detect browser fetchers get real results where plain requests hit "unusual traffic" walls — verified live on Google, DuckDuckGo and Bing. Or plug in your own residential proxy or logged-in session.

📝

New engines without code

A built-in CSS-selector engine means adding a clean-markup search engine is just a JSON file — a result selector plus per-field selectors. DuckDuckGo and Bing are defined this way, no programming required.

⚡

Self-host, no per-query bill

An open-source alternative to hosted SERP APIs: one tiny static binary, no Node, no Docker, no vendor lock-in. Your queries never leave your machine, and every result is checked against frozen real-page fixtures so output stays stable.
